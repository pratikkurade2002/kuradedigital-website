# Kurade Digital Technologies - Website

A modern, production-ready website for Kurade Digital Technologies - a leading software development company.

## 📋 Project Overview

This is a fully responsive, SEO-optimized website built with HTML5, CSS3, Vanilla JavaScript, and Bootstrap 5. It showcases the company's services, products, and provides multiple contact options.

## 📁 Project Structure

```

├── index.html              # Home page

├── js/

├── images/                 # Images directory
└── assets/                 # Additional assets
```

## 🔧 Customization

### Colors
All colors are defined as CSS variables in `style.css`. To change the color scheme:

```css
--primary-color: #1a3a6b;
--secondary-color: #2563eb;
--accent-color: #3b82f6;
```

### Fonts
Font family is defined in the body selector:
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```
